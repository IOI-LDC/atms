# Screen Copy Guide

## Preferred Terms

- Asset
- Package (asset that can contain child assets)
- Component (asset that can be installed in a parent)
- Parent Asset
- Spare (component not currently installed)
- Assembly
- Parts Reference
- Maintenance Request
- Preventive Maintenance
- Corrective Maintenance
- Work Order
- Parts Used
- Usage Reading
- Location History
- Attachments
- ERP Sync
- ERP Reference Data

## Avoided Terms

- Financial Asset
- Depreciation
- Gate Pass
- Chain of Custody
- Logistics workflows or modules, except the fixed `Logistics` user role
- Handover
- Technician Wallet
- Inventory Valuation
- Audit Campaign
- Governance

## Button Labels

Use explicit action labels:

- Create Corrective Request
- Approve & Create Work Order
- Reject Request
- Close Work Order
- Add Reading
- Update Location
- Upload Attachment
- Run ERP Sync

Avoid vague labels:

- Submit
- Process
- Execute
- Authorize Transfer
