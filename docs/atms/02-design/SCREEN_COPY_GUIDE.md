# Screen Copy Guide

## Preferred Terms

- Asset
- Package (asset that can contain child assets)
- Component (asset that can be installed in a parent)
- Parent Asset
- Spare (component not currently installed)
- Assembly
- Parts Management
- Maintenance Request
- Preventive Maintenance
- Corrective Maintenance
- Work Order
- Parts Used
- Usage Reading
- Location History
- Attachments
- ERP Sync
- ERP Reference Data
- Form (WO Form — pre/post-maintenance execution form)
- Pre-maintenance Value
- Post-maintenance Value
- FA Subclass (`fa_subclass_code`)
- FormTemplate (snapshotted into the WO on creation)
- Snapshot (copy of template frozen at WO creation time)
- Sync to Latest (merge WO form with current template version)

## Avoided Terms

- Financial Asset
- Depreciation
- Gate Pass
- Chain of Custody
- Logistics workflows or modules, except the fixed `Logistics` user role
- Handover
- Technician Wallet
- Inventory Valuation
- Audit Campaign
- Governance

## Button Labels

Use explicit action labels:

- Create Corrective Request
- Approve & Create Work Order
- Reject Request
- Close Work Order
- Add Reading
- Update Location
- Upload Attachment
- Run ERP Sync

Avoid vague labels:

- Submit
- Process
- Execute
- Authorize Transfer
